# Create a Kibana space for the bundle

In Kibana, select Management ▶ Stack Management ▶ Kibana: Spaces ▶ Create a space

Suggested space property values:

PropertyValue
NameOMEGAMON Network
InitialsON
IDomegamon-network

TIP

The bundles are designed to be installed in separate spaces.

Bundles might use the same index patterns. If you install bundles in the same space, then the space might contain multiple saved objects for the same index pattern. These duplicates can leading to confusion and versioning issues.
